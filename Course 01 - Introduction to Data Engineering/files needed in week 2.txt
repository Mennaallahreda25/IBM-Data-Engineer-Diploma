IBM Db2 is a popular relational database from IBM that is used for both OLTP and Data Warehousing workloads. The next few labs will guide you to create an instance of Db2 Lite on IBM Cloud. In these labs you will:

Obtain a Feature Code for IBM Cloud

Create an IBM Cloud account using the feature code (so you don't have to provide a credit card)

Create an instance of IBM Db2 Lite

Note that these labs are optional and not required to complete the course. If you are not interested in creating an instance of a Db2 on Cloud database, you may skip to the next lesson on Big Data platforms. If you decide to complete these optional labs, Module 3 also contains optional Db2-based labs.
